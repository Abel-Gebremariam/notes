# Number Row Characters

* ! - Exclamation Mark
* @ - At Sign
* \# - Hashtag/Number Sign
* $ - Dollar Sign
* % - Percent Sign
* ^ - Caret
* & - Ampersand
* \* - Asterisk
* ( - Opening Parenthesis
* ) - Closing Parenthesis
